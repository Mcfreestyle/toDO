# toDO app
![image](https://github.com/Mcfreestyle/toDO/blob/master/toDO.png)

This app manages to-do lists, you can:
* Read, create, update and delete tasks from any list.
* Create to-do lists.
* Complete and undo tasks.
* Login and sign-up.

## Getting started

### How to clone
`$ git clone https://github.com/Mcfreestyle/toDO`

### Install dependences
To backend:
```
/toDO/backend:
$ pip3 install -r requirements.txt
```
To frontend:
```
/toDO/frontend:
$ npm install
```

### Usage
Start API:
```
/toDO/backend:
$ ./start_api.sh
```
Start APP:
```
/toDO/frontend:
$ npm start
```
